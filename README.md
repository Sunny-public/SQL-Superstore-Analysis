# 📊 SQL Data Analysis - Superstore Dataset

## 🔍 Overview

This project demonstrates SQL-based data analysis using the Superstore dataset. The goal is to extract meaningful business insights using structured queries in MySQL.

---

## 🎯 Objectives

* Analyze sales and profit performance
* Identify top-performing categories and customers
* Understand regional trends
* Practice SQL concepts for data analysis

---

## 🛠 Tools Used

* MySQL Workbench
* SQL

---

## 🧠 Skills Demonstrated

* SELECT, WHERE, ORDER BY
* GROUP BY and Aggregations (SUM, AVG)
* Subqueries
* Views
* Indexing

---

## 📊 Key Insights

* Technology category generates the highest sales
* Some sub-categories show consistent losses
* Sales vary significantly across regions
* A small number of customers contribute a large portion of revenue
* Monthly trends show fluctuations in sales performance

---

## 📁 Dataset

* Sample - Superstore dataset (CSV format)

---

## 📸 Screenshots

Screenshots of query outputs are included in the `/SS` folder.

---

## 🚀 How to Run

1. Import the dataset into MySQL
2. Rename table to `superstore`
3. Run queries from `queries.sql`

---

## 💡 Conclusion

This project highlights the importance of SQL in extracting insights from raw data and preparing it for business decision-making.
